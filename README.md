**Assignment 3**

Install Required Libraries
```bash
$ pip install matplotlib
$ pip install torch
$ pip install swig
$ pip install gymnasium[box2d]
```
If `pip install gymnasium[box2d]` gives error in building. Try
```bash
$ pip install wheel setuptools pip --upgrade
$ pip install swig
$ pip install gymnasium[box2d]
```

Run assignment3.ipynb
